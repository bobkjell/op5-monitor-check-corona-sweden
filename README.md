# op5-monitor-check-corona-sweden

A simple check that monitors the Corona-situation in Sweden. <br>
Current metrics are: <br>
* Deaths
* Confirmed
* Hospitalized

## Usage: 
Script takes no argument. Run as it is: <br>
./check_corona_sweden.py

## Requires: 
* python-requests
